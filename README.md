# haruki_helper

## 安装

1. 确保你已经安装了SillyTavern
2. 本项目不区分系统，脚本全平台通用

   ```bash
   git clone https://github.com/saltglow/haruki_helper.git && cd haruki_helper && npm start
   ```

3. 安装完成之后,记得刷新酒馆

---

## 卸载

1. 确保你已经安装了haruki_helper
2. 直接卸载掉整个haruki_helper文件夹.因为本项目没有安装任何依赖,也不会注册任何内容,所以卸载起来比较简单.
